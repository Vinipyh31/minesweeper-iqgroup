# ToDo приложение

Что реализовано:
* Изменение настроек игры
* Список лидеров
* Роутинг
* Фильтрация, сортировка
* Сама игра "сапер"
* Реализовано нажатие на колёсико мыши
* Автоматический выигрыш, если открыты все клетки, на которых нет мин

Используемые технологии:
* JavaScript
* React
* Redux-toolkit
* react-router-dom
* SCSS


## Установка
```
npm install
```


## Запуск
```
npm start
```
